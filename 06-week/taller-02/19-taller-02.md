## Qué hay que hacer?

::contents::
A partir de la **imagen de referencia** que se les suministra, deben recrear la página como una aplicación de **React** usando Vite.

La página debe estar dividida en **componentes reutilizables** y usar **props** y **estado** donde corresponda.

## Paso 1 — Crear el proyecto

::contents::
```bash
# Crear el proyecto con Vite
npm create vite@latest taller-02 -- --template react

# Entrar e instalar dependencias
cd taller-02
npm install

# Iniciar el servidor
npm run dev
```

Limpiar el contenido de `App.jsx` y `App.css` antes de empezar.
